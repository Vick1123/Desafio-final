Sobre mim

Meu nome é Victoria, tenho 23 anos e venho de uma família simples. Embora meu sonho inicial fosse ser musicista, comecei a trabalhar aos 19 anos em diversos empregos, de professora a bartender.
Atualmente, estou seguindo minha paixão na área de tecnologia, cursando Análise e Desenvolvimento de Sistemas para explorar novos horizontes.

Jornada Elas na tech

No início, minha busca por uma carreira na publicidade parecia uma jornada incansável, repleta de portas fechadas e desafios. No entanto, diante das dificuldades, encontrei coragem para explorar um "Plano B" na tecnologia.
Nesse novo território, tentei mergulhar no mundo do UX, mas as mentorias não produziram os resultados esperados.

Foi então que encontrei uma luz no horizonte, representada pela oportunidade chamada "Elas na Tech".

Com esperança e determinação, me inscrevi, abrindo um novo capítulo na minha história. Decidi dar uma segunda chance à programação, abraçando os desafios que vinham com ela. Minha jornada na tecnologia começava, uma aventura repleta de aprendizado e crescimento, onde a busca por meu lugar estava apenas começando.

Desafios Superados

Superar desafios é parte fundamental do crescimento pessoal e profissional.No meu percurso, enfrentei diversos obstáculos que me ajudaram a amadurecer e progredir.

O primeiro deles foi o retorno à programação, depois de ter desistido anteriormente. A perseverança me guiou nessa jornada, e gradualmente recuperei minha confiança.


O segundo desafio consistiu em equilibrar as exigentes aulas com minha rotina diária. A administração do tempo tornou-se essencial para garantir que eu pudesse absorver o máximo de conhecimento possível sem comprometer outros compromissos.
Por fim, o terceiro desafio envolveu o desenvolvimento de projetos e desafios práticos.

Como iniciante, minha base de conhecimento era limitada às aulas e aos materiais compartilhados pela minha mentora, Giu. No entanto, esse desafio me estimulou a explorar e aprender de forma independente, o que enriqueceu minha compreensão da programação.

Conquistas

Conquistar é uma jornada de superação constante. O primeiro marco significativo foi assegurar meu lugar neste programa altamente competitivo. Muitas garotas ambicionavam uma vaga na turma assíncrona, mas com determinação e esforço, conquistei a minha vaga.

No início, duvidei da minha capacidade de enfrentar os desafios que surgiriam pela frente. Contudo, a cada dia e a cada aula, provei para mim mesma que podia superar qualquer obstáculo. Cada desafio que eu superava me impulsionava a continuar, e assim, minhas conquistas só cresciam. Isso me lembra que, com perseverança, podemos alcançar até mesmo o que parecia impossível.

Aprendizados

  • Bootstrap

    Bootstrap é um popular framework de front-end que simplifica o desenvolvimento web, oferecendo uma coleção de estilos e componentes prontos para uso, como botões, formulários e barras de navegação.

  • JavaScript

    JavaScript é uma linguagem de programação de alto nível amplamente utilizada para criar interatividade em páginas da web. Ele permite a manipulação dinâmica de elementos HTML, validação de formulários e muito mais.

  • FLexbox

    Flexbox é um modelo de layout CSS que simplifica a criação de designs flexíveis e responsivos, permitindo o alinhamento e a distribuição de elementos em um contêiner, tornando o design web mais eficiente.

  • Lista Ordenada

    Uma lista ordenada é um elemento HTML que organiza itens em ordem numérica ou alfabética. É frequentemente usado para criar listas numeradas ou com marcadores em documentos web.

  • Menu Hambuerguer

    Um menu hambúrguer é um padrão de design de interface de usuário usado em sites responsivos e aplicativos móveis. Consiste em um ícone de três linhas horizontais empilhadas que, quando clicado, exibe o menu de navegação.

  • Python

    Python é uma linguagem de programação versátil e de fácil leitura, amplamente utilizada no desenvolvimento web, análise de dados, automação, aprendizado de máquina e muito mais.

  • Variáveis

    Variáveis são usadas em programação para armazenar e manipular dados. Elas permitem que os programadores atribuam nomes a valores para referência posterior, tornando o código mais legível e flexível.

  • Doom

    O DOM é uma representação da estrutura de um documento HTML ou XML que permite que os programadores interajam com o conteúdo da página, modificando elementos, estilos e eventos dinamicamente por meio de JavaScript.

Projetos

Projeto 1


Este projeto de página da web utiliza HTML e CSS para criar uma interface atraente. O HTML é responsável pela estrutura da página, definindo cabeçalho, seção de imagens e rodapé. O CSS é usado para estilizar os elementos, como cores de fundo, fontes e posicionamento de elementos. No cabeçalho, o uso de flexbox no elemento pai (.header) permite alinhar o logotipo à esquerda e a lista de navegação à direita. A seção de imagens utiliza flexbox para criar um layout responsivo com imagens centralizadas e espaçadas uniformemente. O efeito de passar o mouse para transformar as imagens em preto e branco é alcançado com a propriedade filter do CSS. Em termos técnicos, este projeto demonstra o uso de HTML e CSS para criar uma página da web interativa e visualmente atraente que pode ser usada como um ponto de partida para explorar mais informações sobre São Paulo por meio dos links de navegação.

Projeto 2


Este projeto de página da web utiliza HTML, CSS e JavaScript para criar uma interface atraente e interativa. O HTML é responsável pela estrutura da página, definindo o cabeçalho, a seção de imagens e o rodapé. O CSS é usado para estilizar os elementos, como cores de fundo e fontes, enquanto o JavaScript adiciona funcionalidade às operações de cálculo na seção principal. No cabeçalho, o uso de flexbox no elemento pai (.header) permite alinhar o logotipo à esquerda e a lista de navegação à direita. A seção de imagens utiliza flexbox para criar um layout responsivo com imagens centralizadas e espaçadas uniformemente. Além disso, o JavaScript é usado para realizar cálculos simples, como adição, subtração, multiplicação e divisão, exibindo os resultados dinamicamente na página. Em termos técnicos, este projeto demonstra a combinação de HTML, CSS e JavaScript para criar uma página da web completa que não apenas exibe informações sobre São Paulo, mas também oferece interatividade por meio de cálculos simples na seção principal.

Projeto 3

Este projeto de página da web utiliza HTML, CSS e JavaScript para criar uma interface atraente e interativa. O HTML é responsável pela estrutura da página, incluindo cabeçalho, seção principal e rodapé. O CSS é utilizado para estilizar os elementos, definindo cores, fontes e posicionamento, enquanto o JavaScript adiciona funcionalidade à página.
No cabeçalho, incorporamos o conceito de menu "hambúrguer", representado por três linhas horizontais empilhadas, que economiza espaço na tela e é comumente usado em sites responsivos e aplicativos móveis. Quando os usuários clicam ou tocam no ícone do menu "hambúrguer", o menu de navegação é exibido de forma expansível, oferecendo acesso fácil às opções de navegação, como links para outras partes do site. Essa abordagem de design é particularmente eficaz em dispositivos móveis, pois permite uma experiência de usuário mais agradável, mantendo a interface limpa e organizada. É um exemplo de como o HTML, CSS e JavaScript podem ser combinados para criar uma página da web atraente e responsiva que se adapta a diferentes tamanhos de tela..
